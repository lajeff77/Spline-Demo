# Spline-Demo

By: Lauryn Jefferson

A demonstration of a Catmull-Rom spline in Java with Slick2D library.

 Controls
 * z key: move to the next node to the right
 * x key: move to the next node to the right
 * up key: move highlighted node up
 * down key: move highlighted node down
 * left key: move highlighted node left
 * right key: move highlighted node right
 * left mouse click: move highlighted node to current mouse position
